### 二、Vue3.x父组件主动获取子组件的数据和执行子组件方法

#### 2.1、调用子组件的时候定义一个ref

```
   <v-header ref="header"></v-header>
```

#### 2.2、父组件主动获取子组件数据

```
this.$refs.header.属性
```

#### 2.3、父组件主动执行子组件方法

```
this.$refs.header.方法
```

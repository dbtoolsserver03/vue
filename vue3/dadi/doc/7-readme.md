### 一、Vue3.x中集成Sass/scsss

#### 1.1、安装sass-loader node-sass

```
npm install -D sass-loader node-sass
```

#### 1.2、style中配置sass/scss

lang可以配置scss ，scoped表示这里写的css只有当前组件有效

```
<style lang = "scss" scoped>
    h2 {
        text-align: center;
    }
</style>
```
